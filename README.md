# NovaGlow Curriculos

Versao do sistema de gestao de curriculos com identidade visual em estilo layout com sidebar lateral.

## Resumo

- Estrutura com menu fixo lateral e conteudo principal separado.
- Cadastro de curriculos com validacao e persistencia local.
- Projeto academico construido com Next.js App Router e Tailwind CSS.

## Rotas principais

- `/`
- `/sistema/paginas/curriculos`
- `/sistema/paginas/curriculos/[id]`
- `/sistema/paginas/curriculos/novo`

## Execucao

```bash
npm install
npm run dev
```
